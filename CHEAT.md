# For windows

## chrome
```
alt + <- // １つ前のページに戻る
alt + shift + B // ヘッダのブックマークバーにフォーカス
ctrl + tab // タブ切替
ctrl + shift + '+' // 拡大
ctrl + '-' // 縮小
ctrl + 0 // 拡大縮小リセット
esc // 操作キャンセル
```

## excel(All of below are ローマ字入力モード)
```
=== Add ===
ctrl + shift + & // 外枠追加
ctrl + alt + V -> V -> Enter // 値だけ貼り付け
ctrl + space -> ctrl + shift + '+' // 列追加
ctrl + shift + L // フィルタON/OFF
F2 // セル入力
shift + F2 // コメント追加
shift + space -> ctrl + shift + '+' // 行追加
=== Remove ===
alt -> A -> C -> 2 // フィルター条件リセット
ctrl + space -> ctrl + shift + '-' // 列削除
shift + space -> ctrl + '-' // 行削除
shift + F2 // コメント削除
=== Other ===
ctrl + PageUp/PageDown // シート左/右移動
ctrl + 1 // セルの書式設定表示
ctrl + F1 // リボンの表示/非表示
esc // 操作キャンセル
```

## google翻訳
```
ctrl + shift + S // 翻訳前/後言語の切替
```

## windows
```
alt + '~' // 入力モード切替
alt + tab // アプリ切替
alt + space -> C // 画面を閉じる
ctrl + win + '←/→' // 仮想ディスプレイ間の移動
ctrl + win + D // 仮想ディスプレイの追加
ctrl + win + F4 // 仮想ディスプレイの削除
esc // 操作キャンセル
win + '←/→' // 画面左右分割
win + '↑/↓' // 画面最大化/最小化
win + tab // 全てのアプリ表示
```
