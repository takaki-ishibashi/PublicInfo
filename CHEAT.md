# For windows

## excel(ローマ字入力モード時)
```
=== 編集系 ===
alt -> H -> B -> A // 格子(田)描画
ctrl + shift + & // 外枠(口)描画
ctrl + alt + V -> V -> Enter // 値だけ貼り付け
ctrl + space -> ctrl + shift + "+" // 列追加
ctrl + shift + L // フィルタON/OFF切り替え
F2 // セル入力開始
shift + F2 // コメント追加
shift + space -> ctrl + shift + "+" // 行追加

=== 削除系 ===
alt -> A -> C -> 2 // フィルタ条件リセット
ctrl + space -> ctrl + shift + "-" // 列削除
shift + space -> ctrl + "-" // 行削除
shift + F2 // コメント削除

=== その他 ===
alt + "↓" // フィルタ条件を選択
ctrl + PageUp/PageDown // シート間移動
ctrl + 1 // セルの書式設定を開く
ctrl + F1 // リボンの表示/非表示切り替え
esc // 操作キャンセル
```

## google chrome
```
alt + '←' // １つ前のページに戻る
alt + shift + B // ヘッダのブックマークバーにカーソルフォーカス
ctrl + tab + [shift] // タブ間移動
ctrl + shift + '+' // 拡大
ctrl + '-' // 縮小
ctrl + 0 // 拡大縮小リセット
ctrl + W // タブを閉じる
esc // 操作キャンセル
```

## google翻訳
```
ctrl + shift + S // 翻訳前言語と翻訳後言語の切り替え
```

## windows10
```
alt + '~' // 入力モード(ひらがな/半角英字)切替
alt + tab // アプリ間の移動
alt + space -> C // 画面を閉じる
ctrl + win + '←/→' // 仮想ディスプレイ間の移動
ctrl + win + D // 仮想ディスプレイの追加
ctrl + win + F4 // 仮想ディスプレイの削除
esc // 操作キャンセル
win + '←/→' // 画面左右分割
win + '↑/↓' // 画面最大化/最小化
win + tab // 全てのアプリ表示
```
